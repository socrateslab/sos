此处的数据收集策略类似于实验研究方法。
- 基于不同的参数将实验分为实验组和对照组，
- 然后让两组模型运行足够多的次数，并记录下来发言者和沉默者人数随时间变化的数据。
- 接下来，需要对通过多主体模型仿真得到的数据进行分析。
